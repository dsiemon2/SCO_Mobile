# Soup Cook Off Mobile

Mobile-first app for soup cook-off events.

**Type:** Mobile App (React + Vite + Capacitor)

## Tech Stack

- **Framework:** React 18.2
- **Build Tool:** Vite 5
- **Language:** TypeScript 5.3
- **Mobile:** Capacitor 5.6
- **CSS:** Tailwind CSS 3.3
- **Animation:** Framer Motion 10.16
- **Icons:** Lucide React
- **Routing:** React Router 6.20

## Ports

| Service | Port | Description |
|---------|------|-------------|
| Static Site | 8150 | Main application |

## Local Development URL

- **App:** http://localhost:8150/

## Docker Setup

```bash
# Start all services
docker compose up -d

# Rebuild and start
docker compose up -d --build

# View logs
docker compose logs -f

# Stop all services
docker compose down
```
